# 

## Enterprise
### ChatGPT Agent
#### tool
- operator: 在網站上捲動、點選和輸入內容
- deep research: 擅於分析和歸納資訊要點
- connector: 連結 Gmail、Github 等應用程式
#### 參與跑分
- 人類終極測驗
- FrontierMath
- DSBench: 評估智慧體在涵蓋資料分析和建模等真實資料科學任務的表現
- SpreadsheetBench: 評估模型在處理源自真實情境的試算表編輯能力
- BrowseComp⁠: 專門評估瀏覽智慧體搜尋網路上難尋資訊的能力
- WebArena: 評估了網頁瀏覽智慧體在完成實際任務方面的表現
