  - 利用 VLM 學習到的統一嵌入空間進行 區域嵌入（regional embedding）。
  - LARE 將影像嵌入在視覺語言嵌入空間中表示為「一個區域」而非單一點，並透過在該區域中取樣影像嵌入來實現數據增強，使得模型能泛化到多個未見領域
